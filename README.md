### Key Insights

* Overall communication is predominantly positive (~70%)
* High-volume communicators tend to have higher sentiment scores
* Negative sentiment often appears in short message bursts
* A decline in sentiment is observed in early 2011

### Recommendations

* Conduct 1:1 check-ins with flagged employees to address concerns
* Recognize and reward consistently positive communicators
* Monitor negative sentiment trends using rolling windows
* Track communication frequency as an early indicator of disengagement
